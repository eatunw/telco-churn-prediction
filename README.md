# telco-churn-prediction

\# Telco Customer Churn Prediction



\## Overview

A machine learning project focused on predicting customer churn in a telecom company and identifying key drivers of customer retention.



\## Dataset

IBM Telco Customer Churn dataset — 7,032 customers, 21 features.



\## Tools Used

Python, pandas, scikit-learn, XGBoost, SHAP, Plotly



\## Key Findings

\- Month-to-month customers churn at 42.7% vs 2.8% for two-year contracts  

\- Nearly 50% of churners leave within the first 12 months  

\- Fiber optic customers churn at 41.9% despite being the premium product  



\## Results

Best model: Logistic Regression — AUC: 0.76, Churn Recall: 0.80
